# Sakthivel — Developer Portfolio

A responsive one-page software developer portfolio built with Next.js, TypeScript, and plain CSS. The visual direction is a dark cinematic landscape with warm accents, glass panels, and product-focused project cards.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

- Personal copy, projects, skills, links, and email: `app/page.tsx`
- Colors, spacing, breakpoints, and visual styling: `app/globals.css`
- Page title and description: `app/layout.tsx`
- Replace `public/sakthi-resume.txt` with your résumé PDF, then update its link in `app/page.tsx`

## Production check

```bash
npm run build
```
